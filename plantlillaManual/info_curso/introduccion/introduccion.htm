Manual de Proceso de consulta Externa

Bienvenido al manual de Proceso de consulta Externa para el sistema SIMA (Sistema Integral Medico Administrativo) en este manual aprenderas a utilizar el sistema para llevar a cabo todo el proceso que conlleva consulta esterna.

Algunas de las cosas que veras son, Configuracion de Horario para los medicos, Agendar Citas, cancelar, cambiar responsable y hacer cargos. Igual aprederas como dar de alta un expediente cunado nuestro paciente viene por primera vez.
